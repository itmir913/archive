#33 м–ҙн”Ңмқҳ Widget(мң„м Ҝ)мқ„ л§Ңл“Өмһҗ

2015л…„ 1мӣ” 5мқј Android/App мӣҗліё ліҙкё° вҶ—

мқҙлІҲ кёҖм—җм„ңлҠ” мң„м Ҝмқ„ н•ңлІҲ л§Ңл“Өм–ҙ ліјк№Ң н•©лӢҲлӢӨ

м•Ҳл…•н•ҳм„ёмҡ”

мқҙлІҲ кёҖм—җм„ңлҠ” мң„м Ҝмқ„ н•ңлІҲ л§Ңл“Өм–ҙ ліјк№Ң н•©лӢҲлӢӨ

м•Ҳл“ңлЎңмқҙл“ңмқҳ мң„м ҜмқҖ TextView, EditTextмҷҖ к°ҷмқҖ мң„м ҜлҸ„ мһҲкі , нҷҲнҷ”л©ҙм—җ лқ„мҡ°лҠ” мң„м ҜлҸ„ мһҲмҠөлӢҲлӢӨ

м—¬кё°м„ңмқҳ мң„м ҜмқҖ нҷҲнҷ”л©ҙм—җ лқ„мҡ°лҠ” мң„м Ҝмқ„ лң»н•©лӢҲлӢӨ

мҷң TextView, EditTextк°Җ мң„м Ҝмқҙлғҗкі  н•ҳлӮҳл©ҙ.. importлҘј ліҙмӢңл©ҙ android.widget.EditTextвҖҰ м•„мӢңкІ мЈ ??

н”„лЎңм қнҠё мғқм„ұ л°Ҹ л Ҳмқҙм•„мӣғ кҙҖл Ё

н”„лЎңм қнҠёмқҳ мқҙлҰ„мқҖ ExampleWidgetмңјлЎң м •н–ҲмҠөлӢҲлӢӨ

н”„лЎңм қнҠёлҘј мІҳмқҢ мғқм„ұн•ҳл©ҙ MainActivityк°Җ мһҲмқ„нғ лҚ°мҡ”

мң„м Ҝмқ„ лӢӨлЈЁлҠ” н”„лЎңм қнҠёмқҙлҜҖлЎң л©”мқё м•ЎнӢ°л№„нӢ°лҠ” кұҙл“Өм§Җ м•ҠмҠөлӢҲлӢӨ

мң„м Ҝм—җ н•„мҡ”н•ң л Ҳмқҙм•„мӣғкіј м•ЎнӢ°л№„нӢ°л§Ң л§Ңл“Ө мҳҲм •мһ…лӢҲлӢӨ

мң„м Ҝмқҳ лӘҮк°Җм§Җ мў…лҘҳ

мң„м Ҝм—җлҸ„ мў…лҘҳк°Җ мһҲлғҗкі  м§Ҳл¬ён•ҳмӢңлҠ” л¶„л“Өмқҙ кі„мӢӨкІғ к°ҷмҠөлӢҲлӢӨ

мӮ¬мӢӨлҢҖлЎң л§җн•ҳл©ҙ мқҙкІғмқҖ м ңк°Җ мһ„мқҳлЎң л¶„лҘҳн•ңкІғмқёлҚ°мҡ”

к·ёлҹј м–ҙл–»кІҢ л¶„лҘҳн–Ҳлғҗл©ҙ..

  1. кё°ліё мң„м Ҝ (мң„м Ҝмқ„ м„ нғқн–Ҳмқ„л•Ң л°”лЎң лЎңл”©лҗЁ)
  2. нҒ¬кё° ліҖкІҪ мң„м Ҝ
  3. м„Өм •мқҙ к°ҖлҠҘн•ң мң„м Ҝ

мқҙл ҮкІҢ 3к°Җм§ҖлЎң лӮҳлҲ лҙӨмҠөлӢҲлӢӨ

мқҙ к°•мўҢм—җм„ңлҠ” 1лІҲкіј 2лІҲмқ„ н•Ёк»ҳ м„ӨлӘ…н•  мҳҲм •мһ…лӢҲлӢӨ

м§ҖкёҲл¶Җн„° мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ

мң„м Ҝмқҳ мһ‘лҸҷ кө¬мЎ°

мң„м Ҝмқҳ кө¬мЎ°лҠ” м•„лһҳмҷҖ к°ҷмҠөлӢҲлӢӨ

мЎ°кёҲ ліөмһЎн•ҙ ліҙмқјмҲҳ мһҲмқ„м§Җ лӘЁлҘҙкІ мҠөлӢҲлӢӨл§Ң..

мҡ°лҰ¬к°Җ л§Ңл“Өм–ҙм•ј н•ҳлҠ” л¶Җл¶„мқҖ developerл¶Җл¶„мһ…лӢҲлӢӨ

мң„м ҜмқҖ м§ҖкёҲк№Ңм§Җ л§Ңл“Өм–ҙмҳЁ м•ұкіјлҠ” лӢӨлҘҙкІҢ нҷҲнҷ”л©ҙм—җм„ң кё°мғқ(?)н•ҳл©ҙм„ң лҸҷмһ‘н•ҳлҠ” кө¬мЎ°мһ…лӢҲлӢӨ

мЎ°кёҲ мқҙн•ҙн•ҳмӢңкё° м–ҙл ӨмӣҢлҸ„ мһҳ л”°лқјмҷҖмЈјм„ёмҡ”

мң„м Ҝмқҳ л Ҳмқҙм•„мӣғмқ„ л§Ңл“Өмһҗ

нҷҲнҷ”л©ҙм—җ лқ„мҡё мң„м ҜлҸ„ л§Ҳм°¬к°Җм§ҖлЎң xmlлЎң л Ҳмқҙм•„мӣғмқ„ м •мқҳн• мҲҳ мһҲмҠөлӢҲлӢӨ

мқҙ к°•мўҢм—җм„ңлҠ” мӢ¬н”Ңн•ҳкІҢ 30л¶„л§ҲлӢӨ мӢңк°„мқ„ лқ„мҡ°лҠ” мң„м Ҝмқ„ л§Ңл“Өм–ҙ ліјк№Ң н•©лӢҲлӢӨ

мқҙл ҮкІҢ мҪ”л“ңлҘј мһ‘м„ұн–ҲмҠөлӢҲлӢӨ

LinearLayoutмқҳ лҶ’мқҙмҷҖ л„Ҳл№„лҘј лӘЁл‘җ match_parentлЎң н•ң мқҙмң лҠ” нҒ¬кё°лҘј мһҗмң лЎӯкІҢ ліҖкІҪн•ҳлҠ” мң„м Ҝ(2лІҲ)мңјлЎң кө¬м„ұн•ҳкё° мң„н•Ёмһ…лӢҲлӢӨ

м—¬кё°м„ң мң„м Ҝм—җ лӘЁл“ л·°к°Җ л“Өм–ҙк°ҲмҲҳ мһҲлҠ”кІғмқҖ м•„лӢҢлҚ°мҡ”

лӢӨмқҢкіј к°ҷмқҖ л·°лҘј м¶”к°Җн•ҙм„ң мӮ¬мҡ©н• мҲҳ мһҲмҠөлӢҲлӢӨ

мң нҳ•л·° мқҙлҰ„
л·° к·ёлЈ№FrameLayout, LinearLayout, RelativeLayout
л·°AnalogClock, Button, Chronometer, ImageButton, ImageView, ProgressBar, TextView

м°ёкі  : Do it м•Ҳл“ңлЎңмқҙл“ң м•ұ н”„лЎңк·ёлһҳл°Қ

мҷң мқҙкІғл“Өл§Ң лҗҳлғҗл©ҙ.. мң„м ҜмңјлЎң н‘ңнҳ„лҗҳлҠ” л·°л“Өмқҙ лӢӨлҘё н”„лЎңм„ёмҠӨм—җ л“Өм–ҙк°Җ мһҲкі ,

мқҙл•Ңл¬ём—җ лӢӨлҘё н”„лЎңм„ёмҠӨмқҳ л·°м—җ м ‘к·јн•ҳкё° мң„н•ҙм„ң RemoteViewsк°Җ мӮ¬мҡ©лҗҳкё° л•Ңл¬ёмһ…лӢҲлӢӨ

RemoteViewsлҠ” м•„лһҳм—җм„ң мӮ¬мҡ©лҗ©лӢҲлӢӨ

м•ұ мң„м Ҝ м ңкіөмһҗ м •ліҙлҘј л§Ңл“Өмһҗ

мқҙ л Ҳмқҙм•„мӣғмқ„ мқҙмҡ©н•ҙм„ң м•ұ мң„м Ҝ м ңкіөмһҗ м •ліҙ(appwidget-provider)лҘј л§Ңл“Өм–ҙм•ј н•©лӢҲлӢӨ

res/xmlнҸҙлҚ”лҘј л§Ңл“Өм–ҙмЈјм„ёмҡ”

нҢҢмқј мқҙлҰ„мқҖ widget_provider_basic.xmlмңјлЎң н•ҳкІ мҠөлӢҲлӢӨ

<?xml version="1.0" encoding="utf-8"?>
<appwidget-provider xmlns:android="http://schemas.android.com/apk/res/android"
    android:initialLayout="@layout/widget_layout"
    android:minHeight="72dp"
    android:minWidth="146dp"
    android:previewImage="@drawable/preview"
    android:resizeMode="horizontal|vertical"
    android:updatePeriodMillis="1800000" />

лӘҮк°Җм§Җ мІҳмқҢліҙлҠ”кІҢ лӮҳмҳӨл„Өмҡ”.. м•„лӢҲ лӘЁл‘җ мІҳмқҢліҙлҠ”кұ° мқјмҲҳлҸ„..?

н•ҳлӮҳм”© мӮҙнҺҙлҙ…мӢңлӢӨ

  • android:initialLayout=вҖң@layout/widget_layoutвҖқ : мң„м Ҝмқҳ мІҳмқҢ л Ҳмқҙм•„мӣғмһ…лӢҲлӢӨ, мІҳмқҢмқҙлқјкі  н•ң мқҙмң лҠ” лӮҳмӨ‘м—җ javaм—җм„ң RemoteViewsлҘј мқҙмҡ©н•ҙ ліҖкІҪн• мҲҳ мһҲкё° л•Ңл¬ёмһ…лӢҲлӢӨ
  • android:minHeight=вҖң72dpвҖқ : мөңмҶҢ лҶ’мқҙ мһ…лӢҲлӢӨ
  • android:minWidth=вҖң146dpвҖқ : мөңмҶҢ л„Ҳл№„ мһ…лӢҲлӢӨ
  • android:previewImage=вҖң@drawable/previewвҖқ : мң„м Ҝмқ„ м„ нғқн• л•Ң лҜёлҰ¬ліҙкё° мқҙлҜём§Җмһ…лӢҲлӢӨ
  • android:resizeMode=вҖңhorizontal|verticalвҖқ : мң„м Ҝмқҳ нҒ¬кё°лҘј ліҖкІҪн• мҲҳ мһҲлҸ„лЎқ м„Өм •н•ҙмӨҚлӢҲлӢӨ
  • android:updatePeriodMillis=вҖң1800000вҖқ : м—…лҚ°мқҙнҠё к°„кІ©мңјлЎң, 1800000лҠ” 30л¶„мһ…лӢҲлӢӨ. 30л¶„ м•„лһҳмқҳ к°„кІ©мңјлЎң м„Өм •н• мҲҳ м—Ҷмңјл©°, 0мңјлЎң м„Өм •н•ҳл©ҙ м—…лҚ°мқҙнҠёлҘј н•ҳм§Җ м•ҠлҠ”лӢӨкі  н•©лӢҲлӢӨ

н•ңлІҲ мқҪм–ҙліҙм…ЁлҠ”лҢҖ м•„м§ҒлҸ„ н’ҖлҰ¬м§Җ м•ҠлҠ”кІҢ мһҲмҠөлӢҲлӢӨ

мҷң лҶ’мқҙк°Җ 72мқёк°Җ..

resizeModeлҠ” л¬ҙм—Үмқҙкі ,, updatePeriodMillisлҠ” 30л¶„мқҙ мөңмҶҢмқёк°Җ??

лЁјм Җ мң„м ҜмқҖ к°ҖлЎң 4м№ё, м„ёлЎң 4м№ёмқҳ л©ҙм Ғмқ„ м°Ём§Җн• мҲҳ мһҲмҠөлӢҲлӢӨ

н•ңм№ёмқ„ 74dpлЎң мһЎлҠ”лҚ°мҡ” мң„м Ҝмқҳ к°ҖмһҘмһҗлҰ¬к°Җ н‘ңмӢңлҗҳлҠ” 2dpлҘј л№јмЈјм–ҙм•ј н•©лӢҲлӢӨ

вҲҙ (мӣҗн•ҳлҠ” м№ё x 74) - 2 dp

ex) 3м№ё : (3 x 74) - 2 = 220dp, 2м№ё : (2 x 74) - 2 = 146dp

resizeModeлҠ” API 12л¶Җн„° мғқкёҙ мҳөм…ҳмһ…лӢҲлӢӨ

horizontal лҳҗлҠ” verticalлҘј м„ нғқн• мҲҳ мһҲкі , к°ҖлЎңм„ёлЎң лӘЁл‘җ к°ҖлҠҘн•ҳкІҢ н•ҳл Өл©ҙ |лҘј л„Јм–ҙмЈјмӢңл©ҙ лҗ©лӢҲлӢӨ

updatePeriodMillisлҠ” м—…лҚ°мқҙнҠё к°„кІ©мһ…лӢҲлӢӨ

30л¶„мқҙ мөңмҶҢ к°„кІ©мқҙл©°, мқҙ м•„лһҳлЎң м„Өм •н•ҙлҸ„ 30л¶„ к°„кІ©мқҙ лҗ©лӢҲлӢӨ

кө¬кёҖм—җм„ң л°°н„°лҰ¬ м Ҳм•ҪмңјлЎң 30л¶„мңјлЎң м„Өм •н•ңкІғмңјлЎң мғқк°Ғлҗ©лӢҲлӢӨ

лҚ” л№ лҘё м—…лҚ°мқҙнҠёлҘј мң„н•ҙм„ңлҠ” м•ҢлһҢмқ„ мқҙмҡ©н•ҳлҠ” л°©лІ•мқҙ мһҲмҠөлӢҲлӢӨ

м•ұ мң„м Ҝ м ңкіөмһҗ нҒҙлһҳмҠӨлҘј л§Ңл“Өмһҗ

AppWidgetProviderлҘј мғҒмҶҚн•ҳлҠ” javaнҢҢмқјмқ„ н•ҳлӮҳ л§Ңл“Өм–ҙмЈјм„ёмҡ”

мқҙлҰ„мқҖ WidgetProviderмңјлЎң н•ҳкІ мҠөлӢҲлӢӨ

public class WidgetProvider extends AppWidgetProvider {

    /**
     * лёҢлЎңл“ңмәҗмҠӨнҠёлҘј мҲҳмӢ н• л•Ң, Overrideлҗң мҪңл°ұ л©”мҶҢл“ңк°Җ нҳём¶ңлҗҳкё° м§Ғм „м—җ нҳём¶ңлҗЁ
     */
    @Override
    public void onReceive(Context context, Intent intent) {
        super.onReceive(context, intent);
    }

    /**
     * мң„м Ҝмқ„ к°ұмӢ н• л•Ң нҳём¶ңлҗЁ
     * 
     * мЈјмқҳ : Configure ActivityлҘј м •мқҳн–Ҳмқ„л•ҢлҠ” мң„м Ҝ л“ұлЎқмӢң мІҳмқҢ н•ңлІҲмқҖ нҳём¶ңмқҙ лҗҳм§Җ м•ҠмҠөлӢҲлӢӨ
     */
    @Override
    public void onUpdate(Context context, AppWidgetManager appWidgetManager,
            int[] appWidgetIds) {
        super.onUpdate(context, appWidgetManager, appWidgetIds);
    }

    /**
     * мң„м Ҝмқҙ мІҳмқҢ мғқм„ұлҗ л•Ң нҳём¶ңлҗЁ
     * 
     * лҸҷмқјн•ң мң„м Ҝмқҙ мғқм„ұлҗҳлҸ„ мөңмҙҲ мғқм„ұл•Ңл§Ң нҳём¶ңлҗЁ
     */
    @Override
    public void onEnabled(Context context) {
        super.onEnabled(context);
    }

    /**
     * мң„м Ҝмқҳ л§Ҳм§Җл§ү мқёмҠӨн„ҙмҠӨк°Җ м ңкұ°лҗ л•Ң нҳём¶ңлҗЁ
     * 
     * onEnabled()м—җм„ң м •мқҳн•ң лҰ¬мҶҢмҠӨ м •лҰ¬н• л•Ң
     */
    @Override
    public void onDisabled(Context context) {
        super.onDisabled(context);
    }

    /**
     * мң„м Ҝмқҙ мӮ¬мҡ©мһҗм—җ мқҳн•ҙ м ңкұ°лҗ л•Ң нҳём¶ңлҗЁ
     */
    @Override
    public void onDeleted(Context context, int[] appWidgetIds) {
        super.onDeleted(context, appWidgetIds);
    }
}

мқҙ мҶҢмҠӨм—җм„ң OverrideлҗҳлҠ” л©”мҶҢл“ңлҠ” м•„лһҳмҷҖ к°ҷмҠөлӢҲлӢӨ

  • public void onReceive(Context context, Intent intent)
  • public void onUpdate(Context context, AppWidgetManager appWidgetManager, int[] appWidgetIds)
  • public void onEnabled(Context context)
  • public void onDisabled(Context context)
  • public void onDeleted(Context context, int[] appWidgetIds)

onReceive()лҠ” лёҢлЎңл“ңмәҗмҠӨнҠёлҰ¬мӢңлІ„м—җм„ң л°°мӣ кё°м—җ л„ҳм–ҙк°ҖкІ мҠөлӢҲлӢӨ

onUpdate()лҠ” мң„м Ҝ к°ұмӢ  мЈјкё°м—җ л”°лқј мң„м Ҝмқ„ к°ұмӢ н• л•Ң нҳём¶ңлҗ©лӢҲлӢӨ

onEnabled()лҠ” мң„м Ҝмқҙ мІҳмқҢ мғқм„ұлҗ л•Ң нҳём¶ңлҗҳл©°, лҸҷмқјн•ң мң„м Ҝмқҳ кІҪмҡ° мІҳмқҢ нҳём¶ңлҗ©лӢҲлӢӨ

onDisabled()лҠ” мң„м Ҝмқҳ л§Ҳм§Җл§ү мқёмҠӨн„ҙмҠӨк°Җ м ңкұ°лҗ л•Ң нҳём¶ңлҗ©лӢҲлӢӨ

onDeleted()лҠ” мң„м Ҝмқҙ мӮ¬мҡ©мһҗм—җ мқҳн•ҙ м ңкұ°лҗ л•Ң нҳём¶ңлҗ©лӢҲлӢӨ

м—¬кё°м„ң м§ҖкёҲ мӮ¬мҡ©н•  л©”мҶҢл“ңлҠ” onUpdate()мһ…лӢҲлӢӨ

мң„м ҜмқҖ мң„м Ҝ idлЎң кө¬л¶„лҗҳлҠ”лҚ°мҡ”

лӘЁл“  мң„м Ҝмқ„ лҸҷмӢңм—җ к°ұмӢ н•ҳкё° мң„н•ҙ м¶”к°Җлҗң лӘЁл“  мң„м Ҝ idлҘј к°Җм ёмҳӨлҠ” мҪ”л“ңлҘј onUpdate()м•Ҳм—җ л„Јм–ҙмӨҚмӢңлӢӨ

public static void updateAppWidget(Context context,
        AppWidgetManager appWidgetManager, int appWidgetId) {
    /**
     * нҳ„мһ¬ мӢңк°„ м •ліҙлҘј к°Җм ёмҳӨкё° мң„н•ң Calendar
     */
    Calendar mCalendar = Calendar.getInstance();
    SimpleDateFormat mFormat = new SimpleDateFormat("yyyy-MM-dd HH:mm",
            Locale.KOREA);

    /**
     * RemoteViewsлҘј мқҙмҡ©н•ҙ Textм„Өм •
     */
    RemoteViews updateViews = new RemoteViews(context.getPackageName(),
            R.layout.widget_layout);

    updateViews.setTextViewText(R.id.mText,
            mFormat.format(mCalendar.getTime()));

    /**
     * л Ҳмқҙм•„мӣғмқ„ нҒҙлҰӯн•ҳл©ҙ нҷҲнҺҳмқҙм§Җ мқҙлҸҷ
     */
    Intent intent = new Intent(Intent.ACTION_VIEW,
            Uri.parse("http://itmir.tistory.com/"));
    PendingIntent pendingIntent = PendingIntent.getActivity(context, 0,
            intent, 0);
    updateViews.setOnClickPendingIntent(R.id.mLayout, pendingIntent);

    /**
     * мң„м Ҝ м—…лҚ°мқҙнҠё
     */
    appWidgetManager.updateAppWidget(appWidgetId, updateViews);
}
