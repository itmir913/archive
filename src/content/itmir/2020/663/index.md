## 서론

안녕하세요.

이번에 소개할 단축어는 사파리에서 사용할 수 있는 구글 번역기 단축어입니다.

![](./images/img.jpg)

먼저 이 단축어의 자바스크립트 부분은 M4XiMUEL님께서 공유해주신 단축어를 일부 참고하였음을 밝힙니다.

원본 출처는 <https://m.clien.net/service/board/pds/12726238> 입니다.

## 단축어 사용법

단축어의 사용법은 간단합니다. 번역을 원하는 대상을 단축어로 넘겨주면 끝입니다.

- 텍스트 : 번역하기를 원하는 텍스트를 드래그한 다음, 공유 버튼을 눌러 이 단축어를 실행해주세요. 텍스트 구글 번역 사이트로 넘어갑니다.

- URL : 사파리를 제외한 다른 앱에서 URL을 공유해주시면, 구글 번역 사이트로 입력받은 URL을 넘겨줍니다.

- 사파리 : 사파리 웹페이지에서 이 단축어를 실행하시면 M4XiMUEL님께서 만드신 구글 번역 자바스크립트가 실행됩니다. 현재 사이트에서 페이지 전환 없이 곧바로 구글 번역이 시작됩니다.

그리고 이 단축어를 단축어 앱에서 곧장 실행할 경우 두 가지 기능을 선택할 수 있습니다.

- 텍스트 입력 : 단축어 앱에서 사용자가 직접 텍스트를 입력한 다음, 입력한 텍스트를 번역할 수 있습니다.

- 번역 언어 선택 : 한국어 이외에 번역 대상 언어를 설정할 수도 있습니다. 설정하지 않으면 기본값은 Korean(ko)입니다.

여기서 설정한 언어는 텍스트, URL, 사파리 번역 모두에 일괄 적용됩니다.

만약 번역 언어를 아랍어로 선택한 다음에 사파리에서 이 단축어를 실행하면, 페이지가 아랍어로 번역됩니다.

만약 번역 대상 언어를 한국어에서 다른 언어로 바꿨다면 그 언어로 페이지가 번역됩니다.
