## 개요

`teacher-workshops` 레포지토리는 교원 연수 및 수업 학습지 Word 문서를 `docx` npm 패키지로 코드에서 생성하는 빌드 시스템이다. `node build.js`로 모든 자료를 일괄 생성한다.

## 구조

```
teacher-workshops/
  연수자료/                    ← 교원 연수 발표용 자료
  학습지자료/                  ← 수업 중 학습지
  build.js                    ← 전체 빌드 엔트리
  package.json                ← docx 의존성
```

## 빌드 방식

`docx` npm 패키지를 사용해 Word 문서를 코드로 생성한다. 문서 내용이 코드로 관리되므로 버전 관리가 가능하고, 여러 문서에 공통 디자인(표, 헤더, 단락 스타일)을 함수로 재사용할 수 있다.

`_Visual_Assets`의 수행평가 docx 생성기(`make_template.js`)와 동일한 `docx` 라이브러리 기반 패턴을 공유한다.
