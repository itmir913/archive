## 개요

학교 현장에서 반복적으로 필요한 작업을 처리하는 소형 도구들을 하나의 레포지토리(`teacher-utility-kit`)에 모아두고 있다. 현재 세 가지 도구가 있다.

---

## score-reporter — 수능 성적 분석 도구

수능 성적 파일(대교협/가채점/김영일 포맷)을 업로드하면 학급별·과목별 통계를 생성한다.

### 주요 설계 결정

**김영일 포맷의 반/번호 없음**: 김영일 스키마에는 `class`, `number` 필드가 없다. 파싱 후 이 값이 빈 문자열인 것은 버그가 아니라 원천 데이터 포맷의 제한이다. 학급별 분석이 불가한 것이 알려진 한계다.

**학년도 `+1` 자동 계산**: 대교협/가채점 스키마에서 `exam_year` 미기입 시 `new Date().getFullYear() + 1`로 설정한다. 수능은 11월에 시행되고 해당 학년도는 다음 해로 표기하는 것이 한국 교육 관행이다(2026년 11월 수능 → 2027학년도).

**모달 외부 클릭 닫기 미구현**: 의도치 않은 클릭으로 모달이 닫히는 것을 방지하기 위해 backdrop 클릭으로 모달을 닫지 않는다. ESC 키만 지원한다.

## code-sheet-builder — 코딩 문제지 제작 도구

프로그래밍 수업용 문제지를 웹 에디터에서 작성하고 PDF로 출력하는 도구다. Monaco 에디터를 사용해서 코드 블록을 직접 편집할 수 있다.

### 주요 수정 이력

- **Monaco 디바운스**: 에디터 변경 시 과도한 렌더링 방지를 위해 디바운스 적용
- **`_maskError` 잔류 버그**: 마스킹된 에러 상태가 지워지지 않던 문제 수정
- **comment 마스크 길이 오류**: 코드 주석 마스킹 시 길이 계산 오류 수정

---

## Prompt Gallery — 교사용 AI 프롬프트 공유

교사들이 AI 도구(ChatGPT, Claude 등)를 활용할 때 유용한 프롬프트를 공유하는 갤러리다. 초기 디렉토리명 `gpts-for-teachers`에서 `prompt-gallery`로 변경됐다.

---

## 공통 원칙

세 도구 모두 아래 원칙을 공유한다:

- **목업 선행**: 목업 완성 후 백엔드/프론트 구현. 목업 없이 구현 시작 금지
- **모달 외부 클릭 닫기 없음**: 모든 도구에서 동일 정책
