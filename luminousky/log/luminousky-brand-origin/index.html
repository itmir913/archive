luminousky — 닉네임의 기원과 도메인

July 12, 2026 · Dev Log

whdghks913에서 itmir913으로, 그리고 luminousky로 — 닉네임 변천사와 뒤늦게 찾아온 도메인

July 12, 2026 Dev Log log

닉네임 변천사

처음 인터넷에서 쓴 아이디는 whdghks913이었다. 의미 없는 조합이었고, 당연히 멋지지 않았다. 2013년 경에 itmir913으로 바꿨다. GitHub 아이디를 포함해 전반적으로 정리했다.

itmir는 IT + Mir다. 당시 닉네임이 Mir(Miru)였고, 블로그 이름이 “미르의 IT 정복기”였다. 블로그 이름에서 자연스럽게 따온 조합이다.

그게 한동안 메인 아이디였다. 그런데 2020년 즈음엔 itmir913도 마음에 들지 않았다. 더 나은 닉네임을 구상하기 시작했다.

어느 순간 luminous와 sky를 합쳤다. luminousky. 빛나는 하늘이라는 느낌이었다.

한 달 차이

만들어놓고 GitHub 계정을 확인했더니 github.com/luminousky가 이미 존재했다. 가입 시점을 보니 내가 이 닉네임을 생각하기 약 한 달 전이었다. 공개 활동은 전혀 없었고, 2026년 현재까지 방치 상태다.

아쉬웠다. 한 달 차이였다.

대안을 생각했다. luminous_ky로 GitHub 아이디를 바꿀까. 그런데 itmir913으로 이미 쌓인 것들이 많았다. 어차피 luminousky를 GitHub에서 쓸 수 없다면, itmir913을 그대로 유지하는 편이 낫다고 결론 냈다. 닉네임은 luminousky로 쓰되, GitHub 아이디는 itmir913으로 남기기로 했다.

itmir와 luminousky

itmir 시절(2012–2021) 블로그에 쓴 글은 657개다. 수준이 낮거나 부끄러운 것도 있지만, 지우기에는 너무 아까운 포스트도 많다. 그래서 Tistory를 거의 안 쓰게 됐어도 계정을 정리하지 못하고 있었다.

거기에 현실적인 우려가 더해졌다. Tistory가 언제 서비스를 중단할지 알 수 없다. 플랫폼이 사라지면 9년치 기록이 함께 사라진다.

이 두 가지 — “지우기엔 아깝다”와 “플랫폼을 믿을 수 없다” — 가 아카이빙을 시작하게 만들었다. 결과가 luminousky.com/archive다. 과거의 아이디 itmir를, 새 브랜드 luminousky 아래에 보존한다.

도메인

2020년 당시엔 도메인까지 생각하지 않았다. 그냥 닉네임이었다.

2025년 8월, AI와 대화하다가 도메인 이야기가 나왔다. 그 자리에서 Cloudflare에서 바로 luminousky.com을 검색해봤다. 구매 가능했다. 즉시 구매했다.

GitHub 아이디는 놓쳤지만 도메인은 가져왔다. 지금은 luminousky.com이 이 모든 작업의 기반이 되고 있다.
